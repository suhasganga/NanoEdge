ØMQ Language Bindings

Bindings provide ØMQ language APIs built on top of the native C API. Anyone can submit a new binding. All contributions to this site fall under the [Terms of Use](/legal:terms-of-use) of this site. See the [guidelines for writing bindings](http://www.zeromq.org/docs:bindings).
