> **isBusinessDay**(`time`): `time is BusinessDay`

Check if a time value is a business day object.

## Parameters[​](#parameters "Direct link to Parameters")

• **time**: [`Time`](/lightweight-charts/docs/4.1/api/type-aliases/Time)

The time to check.

## Returns[​](#returns "Direct link to Returns")

`time is BusinessDay`

`true` if `time` is a [BusinessDay](/lightweight-charts/docs/4.1/api/interfaces/BusinessDay) object, false otherwise.
