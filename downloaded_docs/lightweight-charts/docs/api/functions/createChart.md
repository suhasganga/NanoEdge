> **createChart**(`container`, `options`?): [`IChartApi`](/lightweight-charts/docs/api/interfaces/IChartApi)

This function is the simplified main entry point of the Lightweight Charting Library with time points for the horizontal scale.

## Parameters[​](#parameters "Direct link to Parameters")

• **container**: `string` | `HTMLElement`

ID of HTML element or element itself

• **options?**: [`DeepPartial`](/lightweight-charts/docs/api/type-aliases/DeepPartial) <[`TimeChartOptions`](/lightweight-charts/docs/api/interfaces/TimeChartOptions)>

Any subset of options to be applied at start.

## Returns[​](#returns "Direct link to Returns")

[`IChartApi`](/lightweight-charts/docs/api/interfaces/IChartApi)

An interface to the created chart
