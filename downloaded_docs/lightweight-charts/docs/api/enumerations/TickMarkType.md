Represents the type of a tick mark on the time axis.

## Enumeration Members[​](#enumeration-members "Direct link to Enumeration Members")

### Year[​](#year "Direct link to Year")

> **Year**: `0`

The start of the year (e.g. it's the first tick mark in a year).

---

### Month[​](#month "Direct link to Month")

> **Month**: `1`

The start of the month (e.g. it's the first tick mark in a month).

---

### DayOfMonth[​](#dayofmonth "Direct link to DayOfMonth")

> **DayOfMonth**: `2`

A day of the month.

---

### Time[​](#time "Direct link to Time")

> **Time**: `3`

A time without seconds.

---

### TimeWithSeconds[​](#timewithseconds "Direct link to TimeWithSeconds")

> **TimeWithSeconds**: `4`

A time with seconds.
