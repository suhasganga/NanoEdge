> **isUTCTimestamp**(`time`): `time is UTCTimestamp`

Check if a time value is a UTC timestamp number.

## Parameters[​](#parameters "Direct link to Parameters")

• **time**: [`Time`](/lightweight-charts/docs/5.0/api/type-aliases/Time)

The time to check.

## Returns[​](#returns "Direct link to Returns")

`time is UTCTimestamp`

`true` if `time` is a [UTCTimestamp](/lightweight-charts/docs/5.0/api/type-aliases/UTCTimestamp) number, false otherwise.
