On this page

Represents the type of the last price animation for series such as area or line.

## Enumeration Members[​](#enumeration-members "Direct link to Enumeration Members")

### Disabled[​](#disabled "Direct link to Disabled")

> **Disabled**: `0`

Animation is always disabled

---

### Continuous[​](#continuous "Direct link to Continuous")

> **Continuous**: `1`

Animation is always enabled.

---

### OnDataUpdate[​](#ondataupdate "Direct link to OnDataUpdate")

> **OnDataUpdate**: `2`

Animation is active after new data.
