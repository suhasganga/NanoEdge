##### [saved\_handler::reset](reset.html "saved_handler::reset")

Discard the saved handler, if one exists.

###### [Synopsis](reset.html#beast.ref.boost__beast__saved_handler.reset.synopsis)

```programlisting
bool
reset();
```

###### [Description](reset.html#beast.ref.boost__beast__saved_handler.reset.description)

If `*this`
contains an object, it is destroyed.

###### [Return Value](reset.html#beast.ref.boost__beast__saved_handler.reset.return_value)

`true` if an object was destroyed.
