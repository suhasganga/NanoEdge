##### [websocket::stream::is\_open](is_open.html "websocket::stream::is_open")

Returns `true` if the stream
is open.

###### [Synopsis](is_open.html#beast.ref.boost__beast__websocket__stream.is_open.synopsis)

```programlisting
bool
is_open() const;
```

###### [Description](is_open.html#beast.ref.boost__beast__websocket__stream.is_open.description)

The stream is open after a successful handshake, and when no error has
occurred.
