##### [http::message::key\_compare](key_compare.html "http::message::key_compare")

(Inherited from [`http::basic_fields`](../boost__beast__http__basic_fields.html "http::basic_fields"))

A strictly less predicate for comparing keys, using a case-insensitive
comparison.

###### [Synopsis](key_compare.html#beast.ref.boost__beast__http__message.key_compare.synopsis)

```programlisting
using key_compare = implementation-defined;
```

###### [Description](key_compare.html#beast.ref.boost__beast__http__message.key_compare.description)

The case-comparison operation is defined only for low-ASCII characters.
