##### [http::basic\_fields::key\_compare](key_compare.html "http::basic_fields::key_compare")

A strictly less predicate for comparing keys, using a case-insensitive
comparison.

###### [Synopsis](key_compare.html#beast.ref.boost__beast__http__basic_fields.key_compare.synopsis)

```programlisting
using key_compare = implementation-defined;
```

###### [Description](key_compare.html#beast.ref.boost__beast__http__basic_fields.key_compare.description)

The case-comparison operation is defined only for low-ASCII characters.
