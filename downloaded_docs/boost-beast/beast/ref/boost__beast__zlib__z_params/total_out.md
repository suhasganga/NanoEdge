##### [zlib::z\_params::total\_out](total_out.html "zlib::z_params::total_out")

The total number of bytes output so far.

###### [Synopsis](total_out.html#beast.ref.boost__beast__zlib__z_params.total_out.synopsis)

```programlisting
std::size_t total_out = 0;
```

###### [Description](total_out.html#beast.ref.boost__beast__zlib__z_params.total_out.description)

This field is set by the compression library and must not be updated by
the application.

This field can also be used for statistics or progress reports.
